# New Adhar Portal

A modern, full-stack implementation of the Adhar Portal using Spring Boot and React/Vite.

## Architecture

- **Backend**: Spring Boot 3.2+ (Java 17)
- **Frontend**: React + Vite + Vanilla CSS (Antigravity UI)

## Getting Started

### Backend

1.  Navigate to `backend`.
2.  Run `mvn spring-boot:run`.
3.  Server starts at `http://localhost:8080`.

### Frontend

1.  Navigate to `frontend`.
2.  Run `npm install` (first time only).
3.  Run `npm run dev`.
4.  App starts at `http://localhost:5173`.
