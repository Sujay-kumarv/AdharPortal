# Adhar Portal Demo
A demonstration Adhar Portal connected to a mock Qlink service.

## Project Overview
This project simulates a secure identity portal with the following features:
- **Login/Registration**: Secure access simulation.
- **Dashboard**: View user details.
- **Qlink Integration**: Simulates connection to an external 'Qlink' service for status checks.

## How to Run
### Prerequisites
- Java 17 or higher (Recommended).
- Maven is included locally in `tools/`.

### Running the App
1. **One-Click**: Double-click `run.bat` in this folder.
2. **Command Line**:
   ```cmd
   tools\apache-maven-3.9.6\bin\mvn.cmd spring-boot:run
   ```
